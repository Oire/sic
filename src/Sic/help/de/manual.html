Überblick

SIC! ist ein einfacher und barrierefreier Bildformat-Konverter. Er ermöglicht das Konvertieren von Bildern zwischen gängigen Formaten, optional die Größenänderung und die Verarbeitung mehrerer Bilder auf einmal.

Hauptfenster

Das Hauptfenster besteht aus folgenden Bereichen:

Menüleiste
Vier Menüs am oberen Rand des Fensters:
  • Datei — Bilder hinzufügen (aus Dateien, Ordnern oder per Link), Einstellungen öffnen oder die Anwendung beenden.
  • Bearbeiten — Ausgewählte oder alle Bilder aus der Warteschlange entfernen. Dieses Menü ist deaktiviert, wenn die Warteschlange leer ist.
  • Konvertieren — Konvertierung starten oder ein Mehrgrößen-ICO erstellen. Dieses Menü ist deaktiviert, wenn die Warteschlange leer ist.
  • Hilfe — Dieses Handbuch öffnen, nach Softwareaktualisierungen suchen, spenden oder den Info-Dialog anzeigen.
Bildliste
Die linke Hälfte des Fensters zeigt die Warteschlange als Tabelle mit den Spalten: Dateiname, Format, Abmessungen, Größe und Status.
Vorschaubereich
Die rechte Hälfte des Fensters zeigt eine Vorschau des aktuell ausgewählten Bildes.
Steuerungsbereich
Unterhalb der Bildliste und Vorschau:
  • Zielformat — Dropdown-Liste zur Auswahl des Ausgabeformats.
  • Größe ändern — Kontrollkästchen, bei Aktivierung erscheinen der Größenänderungsmodus (Proportionen beibehalten oder Zuschneiden) und die Felder Breite/Höhe.
Konvertierungsschaltflächen
Ausgewählte konvertieren konvertiert nur das markierte Bild; Alle konvertieren verarbeitet jedes Bild in der Warteschlange.
Statusleiste
Am unteren Rand des Fensters werden Statusmeldungen wie Konvertierungsfortschritt und -ergebnisse angezeigt.

Installationsmodi

SIC! kann in zwei Modi ausgeführt werden:

Installer (Standard)
Bei normaler Installation speichert SIC! seine Konfiguration, konvertierte Dateien und Protokolle in %APPDATA%\Oire\Sic\. Der Standard-Ausgabeordner ist %APPDATA%\Oire\Sic\Converted\.
Portabel
Wenn neben der ausführbaren Datei ein userdata-Ordner existiert, läuft SIC! im portablen Modus. Alle Daten (Konfiguration, konvertierte Dateien, Protokolle) werden in diesem userdata-Ordner anstatt in %APPDATA% gespeichert. So können Sie SIC! von einem USB-Stick oder einem beliebigen Ordner ausführen, ohne Spuren im System zu hinterlassen.

Bilder hinzufügen

Es gibt mehrere Möglichkeiten, Bilder zur Warteschlange hinzuzufügen:

Dateien öffnen
Verwenden Sie Datei > Bild hinzufügen (Strg+N), um einen Dateidialog zu öffnen. Sie können mehrere Dateien gleichzeitig auswählen.
Ordner hinzufügen
Verwenden Sie Datei > Ordner hinzufügen (Strg+Umschalt+N), um alle Bilder aus einem Ordner hinzuzufügen. Sie können auswählen, welche Dateitypen einbezogen werden und ob Unterordner durchsucht werden sollen.
Per Link hinzufügen
Verwenden Sie Datei > Bild per Link hinzufügen (Strg+L), um ein Bild von einer URL herunterzuladen und zur Warteschlange hinzuzufügen.
Drag and Drop
Ziehen Sie Bilddateien aus dem Datei-Explorer und legen Sie sie auf der Bildliste ab.
Aus Zwischenablage einfügen
Drücken Sie Strg+V zum Einfügen. SIC! unterstützt das Einfügen von Dateien (z. B. aus dem Datei-Explorer oder Outlook kopiert), Bitmap-Rohdaten (z. B. von Bildschirmaufnahmen) und URLs aus der Zwischenablage.

Bilder konvertieren

  1. Fügen Sie ein oder mehrere Bilder zur Warteschlange hinzu.
  2. Wählen Sie das gewünschte Zielformat aus der Dropdown-Liste Zielformat.
  3. Klicken Sie auf Alle konvertieren (Strg+Umschalt+F5), um jedes Bild in der Warteschlange zu konvertieren, oder wählen Sie bestimmte Bilder aus und klicken Sie auf Ausgewählte konvertieren (F5).

Standardmäßig werden konvertierte Dateien im Ordner Converted innerhalb des Anwendungsdatenverzeichnisses gespeichert. Sie können dies in den Einstellungen ändern.

Wenn bereits eine Datei mit dem gleichen Namen existiert, fragt SIC!, ob Sie überschreiben, umbenennen (durch Hinzufügen eines _1-Suffixes) oder die Datei überspringen möchten.

Größe ändern

Aktivieren Sie das Kontrollkästchen Größe ändern, um die Größenänderung bei der Konvertierung zu aktivieren. Zwei Modi sind verfügbar:

Proportionen beibehalten (Standard)
Geben Sie eine Breite, eine Höhe oder beides ein. Das Bild wird unter Beibehaltung des Seitenverhältnisses skaliert. Wenn Sie nur eine Abmessung eingeben, wird die andere automatisch berechnet.
Zuschneiden
Sowohl Breite als auch Höhe sind erforderlich. Das Bild wird so skaliert, dass es die Zielabmessungen abdeckt, und dann auf die exakte Größe zugeschnitten.

Wenn Sie ein Bild in der Liste bei aktivierter Größenänderung auswählen, werden die Breite- und Höhefelder mit den aktuellen Abmessungen des Bildes vorausgefüllt.

Mehrgrößen-ICO-Dateien erstellen

Um eine ICO-Datei mit mehreren Größen zu erstellen (nützlich für Anwendungssymbole):

  1. Wählen Sie ein Bild in der Liste aus.
  2. Wählen Sie Konvertieren > Mehrgrößen-ICO erstellen (Strg+Alt+F5).
  3. Wählen Sie eine Voreinstellung oder konfigurieren Sie benutzerdefinierte Größen im erscheinenden Dialog.

Folgende Voreinstellungen sind verfügbar:

Favicon
Erzeugt die Größen 16, 32, 48 und 64 Pixel. Ein Favicon ist das kleine Symbol, das in Browser-Tabs, Lesezeichen und der Adressleiste angezeigt wird. Diese Voreinstellung deckt alle Größen ab, die üblicherweise von Webbrowsern erwartet werden.
Anwendungssymbol
Erzeugt die Größen 16, 20, 24, 32, 40, 48, 64 und 256 Pixel. Dies deckt den gesamten Größenbereich ab, den Windows für Anwendungssymbole verwendet — von kleinen Taskleistensymbolen bis zu den großen Kacheln im Explorer.
Benutzerdefiniert
Ermöglicht das Erstellen einer eigenen Größenliste. Startet mit der zuvor ausgewählten Voreinstellung als Basis, dann können Sie einzelne Größen hinzufügen oder entfernen.

Einstellungen

Öffnen Sie Datei > Einstellungen (Strg+,) zum Konfigurieren:

Ausgabeordner
Legen Sie einen benutzerdefinierten Ordner fest, in dem alle konvertierten Dateien gespeichert werden. Klicken Sie auf Zurücksetzen, um den Standard-Ordner Converted innerhalb des Anwendungsdatenverzeichnisses wiederherzustellen.
Sprache
Wählen Sie die Sprache der Benutzeroberfläche. Wählen Sie System, um der Windows-Spracheinstellung zu folgen.
Beenden bestätigen, wenn die Warteschlange nicht leer ist
Wenn aktiviert (Standardeinstellung), fragt SIC! vor dem Schließen nach einer Bestätigung, sofern sich noch Bilder in der Warteschlange befinden. Deaktivieren Sie diese Option, um ohne Nachfrage zu beenden.

Einstellungen werden in Sic.cfg innerhalb des Anwendungsdatenverzeichnisses gespeichert (den genauen Speicherort finden Sie unter Installationsmodi).

Softwareaktualisierungen

SIC! kann automatisch nach neuen Versionen suchen. Eine Aktualisierungsprüfung erfolgt beim Start und dann alle 24 Stunden, solange die Anwendung läuft. Sie können auch manuell über Hilfe > Nach Updates suchen prüfen.

Wenn eine Aktualisierung verfügbar ist, zeigt SIC! einen Dialog mit der neuen Versionsnummer an. Sie können wählen, ob Sie aktualisieren, diese Version überspringen oder später erinnert werden möchten. Wenn Sie auf Aktualisieren klicken, wird die neue Version automatisch heruntergeladen. Nach Abschluss des Downloads erscheint ein Bestätigungsdialog mit den Schaltflächen Installieren und Abbrechen. Durch Klicken auf Installieren wird die Installation gestartet.

Kommandozeilenverwendung

SIC! kann auch über die Kommandozeile ohne Öffnen der grafischen Oberfläche verwendet werden. Dies ist nützlich für Skripte oder Stapelverarbeitung.

sic -i input.png -o output.jpg
sic --input photo.bmp --output photo.webp
sic -i avatar.png -o avatar.ico --resize 128x128
sic -i photo.jpg -f png --resize 800x
--input, -i (erforderlich)
Pfad zum Quellbild.
--output, -o
Pfad zur Ausgabedatei. Das Zielformat wird aus der Dateierweiterung abgeleitet.
--format, -f
Name des Zielformats (z. B. png, jpg). Verwenden Sie dies anstelle von --output, um den Ausgabepfad automatisch zu generieren.
--resize, -r
Bildgröße ändern. Format: BxH, Bx (nur Breite) oder xH (nur Höhe).
--crop, -c
Zuschneiden statt proportional skalieren. Erfordert sowohl Breite als auch Höhe in --resize.

Es muss entweder --output oder --format angegeben werden (aber nicht beides).

Fehlerbehebung

Eine Cloud-Datei kann nicht geladen werden
Dateien, die in OneDrive oder SharePoint gespeichert und nicht lokal heruntergeladen sind, erscheinen als Cloud-Platzhalter. SIC! erkennt dies und fragt, ob Sie sie zuerst herunterladen möchten.
Konvertierung schlägt bei einer bestimmten Datei fehl
Überprüfen Sie, ob die Datei ein gültiges Bild ist und nicht beschädigt ist. SIC! protokolliert detaillierte Fehler im Ordner logs innerhalb des Anwendungsdatenverzeichnisses.
Der Ausgabeordner existiert nicht mehr
Wenn der benutzerdefinierte Ausgabeordner gelöscht oder verschoben wurde, warnt SIC! Sie und verwendet den Standard-Ordner Converted.
