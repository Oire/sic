Présentation

SIC! est un convertisseur de formats d'images simple et accessible. Il vous permet de convertir des images entre les formats populaires, de les redimensionner si nécessaire et de traiter plusieurs images à la fois.

Fenêtre principale

La fenêtre principale se compose des zones suivantes :

Barre de menus
Quatre menus en haut de la fenêtre :
  • Fichier — ajouter des images (depuis des fichiers, dossiers ou liens), ouvrir les paramètres ou quitter l'application.
  • Édition — supprimer les images sélectionnées ou toutes les images de la file. Ce menu est désactivé lorsque la file est vide.
  • Convertir — lancer la conversion ou créer un ICO multi-tailles. Ce menu est désactivé lorsque la file est vide.
  • Aide — ouvrir ce manuel, faire un don ou afficher la boîte de dialogue À propos.
Liste des images
La moitié gauche de la fenêtre affiche la file d'attente sous forme de tableau avec les colonnes : Nom du fichier, Format, Dimensions, Taille et Statut.
Panneau d'aperçu
La moitié droite de la fenêtre affiche un aperçu de l'image actuellement sélectionnée.
Zone de contrôle
Sous la liste des images et l'aperçu :
  • Format cible — liste déroulante pour choisir le format de sortie.
  • Redimensionner — case à cocher qui, une fois activée, affiche le mode de redimensionnement (Garder les proportions ou Recadrer) et les champs Largeur/Hauteur.
Boutons de conversion
Convertir la sélection convertit uniquement l'image mise en surbrillance ; Tout convertir traite toutes les images de la file.
Barre d'état
En bas de la fenêtre, affiche les messages d'état tels que la progression et les résultats de la conversion.

Modes d'installation

SIC! peut fonctionner selon deux modes :

Installateur (par défaut)
Lors d'une installation normale, SIC! stocke sa configuration, les fichiers convertis et les journaux dans %APPDATA%\Oire\Sic\. Le dossier de sortie par défaut est %APPDATA%\Oire\Sic\Converted\.
Portable
Si un dossier userdata existe à côté de l'exécutable, SIC! fonctionne en mode portable. Toutes les données (configuration, fichiers convertis, journaux) sont stockées dans ce dossier userdata au lieu de %APPDATA%. Cela vous permet d'exécuter SIC! depuis une clé USB ou n'importe quel dossier sans laisser de traces sur le système.

Ajouter des images

Il existe plusieurs façons d'ajouter des images à la file :

Ouvrir des fichiers
Utilisez Fichier > Ajouter une image (Ctrl+N) pour ouvrir un sélecteur de fichiers. Vous pouvez sélectionner plusieurs fichiers à la fois.
Ajouter un dossier
Utilisez Fichier > Ajouter un dossier (Ctrl+Maj+N) pour ajouter toutes les images d'un dossier. Vous pouvez choisir les types de fichiers à inclure et s'il faut parcourir les sous-dossiers.
Ajouter par lien
Utilisez Fichier > Ajouter une image par lien (Ctrl+L) pour télécharger une image depuis une URL et l'ajouter à la file.
Glisser-déposer
Faites glisser des fichiers image depuis l'Explorateur de fichiers et déposez-les sur la liste des images.
Coller depuis le presse-papiers
Appuyez sur Ctrl+V pour coller. SIC! prend en charge le collage de fichiers (par exemple copiés depuis l'Explorateur de fichiers ou Outlook), de données bitmap brutes (par exemple depuis Impr. écran) et d'URL depuis le presse-papiers.

Convertir des images

  1. Ajoutez une ou plusieurs images à la file.
  2. Sélectionnez le format cible souhaité dans la liste déroulante Format cible.
  3. Cliquez sur Tout convertir (Ctrl+Maj+F5) pour convertir toutes les images de la file, ou sélectionnez des images spécifiques et cliquez sur Convertir la sélection (F5).

Par défaut, les fichiers convertis sont enregistrés dans un dossier Converted à l'intérieur du répertoire de données de l'application. Vous pouvez modifier cela dans les Paramètres.

Si un fichier portant le même nom existe déjà, SIC! vous demandera si vous souhaitez remplacer, renommer (en ajoutant un suffixe _1) ou ignorer ce fichier.

Redimensionnement

Cochez la case Redimensionner pour activer le redimensionnement lors de la conversion. Deux modes sont disponibles :

Garder les proportions (par défaut)
Saisissez une largeur, une hauteur ou les deux. L'image sera mise à l'échelle en conservant ses proportions. Si vous ne saisissez qu'une dimension, l'autre est calculée automatiquement.
Recadrer
La largeur et la hauteur sont toutes deux requises. L'image est redimensionnée pour couvrir les dimensions cibles, puis recadrée à la taille exacte.

Lorsque vous sélectionnez une image dans la liste avec le redimensionnement activé, les champs de largeur et de hauteur sont pré-remplis avec les dimensions actuelles de l'image.

Créer des fichiers ICO multi-tailles

Pour créer un fichier ICO contenant plusieurs tailles (utile pour les icônes d'application) :

  1. Sélectionnez une image dans la liste.
  2. Choisissez Convertir > Créer un ICO multi-tailles (Ctrl+Alt+F5).
  3. Choisissez un préréglage ou configurez des tailles personnalisées dans la boîte de dialogue qui apparaît.

Les préréglages suivants sont disponibles :

Favicon
Génère les tailles 16, 32, 48 et 64 pixels. Un favicon est la petite icône affichée dans les onglets du navigateur, les favoris et la barre d'adresse. Ce préréglage couvre toutes les tailles généralement attendues par les navigateurs web.
Icône d'application
Génère les tailles 16, 20, 24, 32, 40, 48, 64 et 256 pixels. Cela couvre toute la gamme de tailles utilisées par Windows pour les icônes d'application — des petites icônes de la barre des tâches aux grandes vignettes affichées dans l'Explorateur.
Personnalisé
Permet de créer votre propre liste de tailles. Commence avec le préréglage précédemment sélectionné comme base, puis vous pouvez ajouter ou supprimer des tailles individuelles.

Paramètres

Ouvrez Fichier > Paramètres (Ctrl+,) pour configurer :

Dossier de sortie
Définissez un dossier personnalisé où tous les fichiers convertis seront enregistrés. Cliquez sur Réinitialiser pour restaurer le dossier par défaut Converted à l'intérieur du répertoire de données de l'application.
Langue
Choisissez la langue de l'interface. Sélectionnez Système pour suivre le paramètre de langue de Windows.

Les paramètres sont stockés dans Sic.cfg à l'intérieur du répertoire de données de l'application (voir Modes d'installation pour l'emplacement exact).

Utilisation en ligne de commande

SIC! peut également être utilisé depuis la ligne de commande sans ouvrir l'interface graphique. C'est utile pour les scripts ou le traitement par lots.

sic -i input.png -o output.jpg
sic --input photo.bmp --output photo.webp
sic -i avatar.png -o avatar.ico --resize 128x128
sic -i photo.jpg -f png --resize 800x
--input, -i (obligatoire)
Chemin de l'image source.
--output, -o
Chemin du fichier de sortie. Le format cible est déduit de l'extension du fichier.
--format, -f
Nom du format cible (par exemple png, jpg). Utilisez ceci au lieu de --output pour générer automatiquement le chemin de sortie.
--resize, -r
Redimensionner l'image. Format : LxH, Lx (largeur uniquement) ou xH (hauteur uniquement).
--crop, -c
Recadrer au lieu de mettre à l'échelle proportionnellement. Nécessite à la fois la largeur et la hauteur dans --resize.

Il faut fournir soit --output, soit --format (mais pas les deux).

Dépannage

Un fichier dans le cloud ne peut pas être chargé
Les fichiers stockés dans OneDrive ou SharePoint qui n'ont pas été téléchargés localement apparaissent comme des espaces réservés dans le cloud. SIC! le détectera et vous demandera si vous souhaitez d'abord les télécharger.
La conversion échoue pour un fichier spécifique
Vérifiez que le fichier est une image valide et non corrompue. SIC! enregistre les erreurs détaillées dans le dossier logs à l'intérieur du répertoire de données de l'application.
Le dossier de sortie n'existe plus
Si le dossier de sortie personnalisé a été supprimé ou déplacé, SIC! vous avertira et utilisera le dossier par défaut Converted.
