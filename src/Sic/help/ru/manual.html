Обзор

SIC! — это простой и доступный конвертер форматов изображений. Он позволяет конвертировать изображения между популярными форматами, при необходимости изменять их размер и обрабатывать несколько изображений одновременно.

Главное окно

Главное окно состоит из следующих областей:

Строка меню
Четыре меню в верхней части окна:
  • Файл — добавление изображений (из файлов, папок или по ссылке), открытие настроек или выход из приложения.
  • Правка — удаление выбранных или всех изображений из очереди. Это меню неактивно, когда очередь пуста.
  • Конвертация — запуск конвертации или создание многоразмерного ICO. Это меню неактивно, когда очередь пуста.
  • Справка — открытие данного руководства, проверка обновлений, пожертвование или просмотр диалога «О программе».
Список изображений
Левая половина окна отображает очередь обработки в виде таблицы со столбцами: Имя файла, Формат, Размеры, Размер файла и Статус.
Панель предварительного просмотра
Правая половина окна отображает предварительный просмотр выбранного изображения.
Область элементов управления
Под списком изображений и панелью просмотра:
  • Целевой формат — выпадающий список для выбора формата.
  • Изменить размер — флажок, при установке которого появляются режим изменения размера (Сохранять пропорции или Обрезка) и поля Ширина/Высота.
Кнопки конвертации
Конвертировать выбранные конвертирует только выделенное изображение; Конвертировать все обрабатывает все изображения в очереди.
Строка состояния
В нижней части окна отображаются сообщения о состоянии, такие как ход и результаты конвертации.

Режимы установки

SIC! может работать в двух режимах:

Установка (по умолчанию)
При обычной установке SIC! хранит конфигурацию, сконвертированные файлы и журналы в %APPDATA%\Oire\Sic\. Папка для сохранения по умолчанию — %APPDATA%\Oire\Sic\Converted\.
Портативный режим
Если рядом с исполняемым файлом существует папка userdata, SIC! работает в портативном режиме. Все данные (конфигурация, сконвертированные файлы, журналы) хранятся в этой папке userdata вместо %APPDATA%. Это позволяет запускать SIC! с USB-накопителя или из любой папки, не оставляя следов в системе.

Добавление изображений

Существует несколько способов добавить изображения в очередь:

Открыть файлы
Используйте Файл > Добавить изображение (Ctrl+N), чтобы открыть окно выбора файлов. Можно выбрать несколько файлов одновременно.
Добавить папку
Используйте Файл > Добавить папку (Ctrl+Shift+N), чтобы добавить все изображения из папки. Можно выбрать типы файлов и включить поиск в подпапках.
Добавить по ссылке
Используйте Файл > Добавить изображение по ссылке (Ctrl+L), чтобы загрузить изображение по URL-адресу и добавить его в очередь.
Перетаскивание
Перетащите файлы изображений из Проводника и отпустите их на список изображений.
Вставка из буфера обмена
Нажмите Ctrl+V для вставки. SIC! поддерживает вставку файлов (например, скопированных из Проводника или Outlook), растровых данных (например, после нажатия Print Screen) и URL-адресов из буфера обмена.

Конвертация изображений

  1. Добавьте одно или несколько изображений в очередь.
  2. Выберите нужный целевой формат из выпадающего списка Целевой формат.
  3. Нажмите Конвертировать все (Ctrl+Shift+F5), чтобы конвертировать все изображения в очереди, или выберите нужные изображения и нажмите Конвертировать выбранные (F5).

По умолчанию сконвертированные файлы сохраняются в папку Converted внутри каталога данных приложения. Вы можете изменить это в Настройках.

Если файл с таким именем уже существует, SIC! предложит выбор: перезаписать, переименовать (добавив суффикс _1) или пропустить этот файл.

Изменение размера

Установите флажок Изменить размер, чтобы включить изменение размера при конвертации. Доступны два режима:

Сохранять пропорции (по умолчанию)
Введите ширину, высоту или оба значения. Изображение будет масштабировано с сохранением пропорций. Если указано только одно измерение, второе вычисляется автоматически.
Обрезка
Требуются и ширина, и высота. Изображение масштабируется до покрытия целевых размеров, а затем обрезается до точного размера.

Когда вы выбираете изображение в списке при включённом изменении размера, поля ширины и высоты автоматически заполняются текущими размерами изображения.

Создание многоразмерных ICO-файлов

Чтобы создать ICO-файл с несколькими размерами (полезно для значков приложений):

  1. Выберите изображение в списке.
  2. Выберите Конвертация > Создать многоразмерный ICO (Ctrl+Alt+F5).
  3. Выберите предустановку или настройте пользовательские размеры в появившемся диалоге.

Доступны следующие предустановки:

Favicon
Создаёт размеры 16, 32, 48 и 64 пикселя. Favicon — это маленький значок, отображаемый во вкладках браузера, закладках и адресной строке. Эта предустановка охватывает все размеры, которые обычно ожидают веб-браузеры.
Значок приложения
Создаёт размеры 16, 20, 24, 32, 40, 48, 64 и 256 пикселей. Это охватывает весь диапазон размеров, используемых Windows для значков приложений — от маленьких значков на панели задач до крупных плиток в Проводнике.
Пользовательский
Позволяет составить собственный список размеров. За основу берётся ранее выбранная предустановка, после чего можно добавлять или удалять отдельные размеры.

Настройки

Откройте Файл > Настройки (Ctrl+,) для настройки:

Папка сохранения
Задайте пользовательскую папку, в которую будут сохраняться все сконвертированные файлы. Нажмите Сбросить, чтобы вернуть папку Converted по умолчанию внутри каталога данных приложения.
Язык
Выберите язык интерфейса. Выберите System, чтобы использовать язык системы Windows.
Подтверждать выход при непустой очереди
Если включено (по умолчанию), SIC! запрашивает подтверждение перед закрытием, если в очереди остались изображения. Отключите эту опцию, чтобы выходить без запроса.

Настройки хранятся в файле Sic.cfg внутри каталога данных приложения (точное расположение см. в разделе Режимы установки).

Обновление программы

SIC! может автоматически проверять наличие новых версий. Проверка выполняется при запуске, а затем каждые 24 часа, пока приложение работает. Вы также можете проверить вручную через Справка > Проверить наличие обновлений.

Когда доступно обновление, SIC! показывает диалоговое окно с номером новой версии. Вы можете выбрать: обновить, пропустить эту версию или получить напоминание позже. При нажатии на Обновить новая версия загружается автоматически. После завершения загрузки появляется диалоговое окно подтверждения с кнопками Установить и Отмена. Нажатие на Установить запускает установку.

Использование из командной строки

SIC! можно также использовать из командной строки без открытия графического интерфейса. Это удобно для скриптов и пакетной обработки.

sic -i input.png -o output.jpg
sic --input photo.bmp --output photo.webp
sic -i avatar.png -o avatar.ico --resize 128x128
sic -i photo.jpg -f png --resize 800x
--input, -i (обязательный)
Путь к исходному изображению.
--output, -o
Путь к выходному файлу. Целевой формат определяется по расширению файла.
--format, -f
Название целевого формата (например, png, jpg). Используйте вместо --output для автоматического создания пути.
--resize, -r
Изменить размер изображения. Формат: ШxВ, Шx (только ширина) или xВ (только высота).
--crop, -c
Обрезать вместо пропорционального масштабирования. Требуются и ширина, и высота в --resize.

Необходимо указать либо --output, либо --format (но не оба).

Устранение неполадок

Облачный файл не удаётся загрузить
Файлы, хранящиеся в OneDrive или SharePoint и не загруженные локально, отображаются как облачные заполнители. SIC! обнаружит это и предложит сначала загрузить их.
Конвертация не удаётся для определённого файла
Убедитесь, что файл является корректным изображением и не повреждён. SIC! записывает подробные ошибки в папку logs внутри каталога данных приложения.
Папка сохранения больше не существует
Если пользовательская папка для сохранения была удалена или перемещена, SIC! предупредит об этом и будет использовать папку Converted по умолчанию.
